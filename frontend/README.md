# Sphere Frontend

This frontend is built with Vite, React, and TypeScript.

## Setup

1. Open a terminal in `frontend`
2. Run `npm install`
3. Run `npm run dev` to launch the development server

## Build

Run `npm run build` to produce a production-ready build.

## API proxy

The dev server proxies `/api` requests to `http://localhost:8080` so the frontend can communicate with the Sphere backend during development.
